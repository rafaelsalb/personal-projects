A sorting algorithm is used to organize data according to a user-defined criteria. Bubble sort is a very simple sorting algorithm, usually used for educational purposes only, as it is very slow at sorting large sets of data, but very easily understandable.

Bubble sort works by comparing an element to its predecessor, and swapping their positions if the element is smaller. This idea is repeated for every pair of consecutive elements in a array, until no swaps happen, which means the elements are sorted.
